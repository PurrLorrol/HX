# 我只是来修电脑的... 🔧

> 一个关于“勇者”与“修电脑”的搞笑校园RPG

🎮 **在线玩**：https://purrlorrol.github.io/HX/    https://hx-sigma.vercel.app/

---

## 📖 故事背景

本来，我只是被班主任叫去修电脑……  
结果一进教学楼，画风就变了！

走廊上滚着会攻击人的篮球，教室里有会飞的词典，天台上还有挥着拖把的“魔王”后勤主任……

更离谱的是，我遇到了三个自称“勇者”的同学：
- 班长（重度中二病）
- 学妹（想当魔法少女）
- 学霸（只想回去写作业）

他们负责喊口号，我负责……修电脑。

## 🎮 游戏特色

- ⏱️ **短小精悍**：全流程 10–15 分钟，适合课间或摸鱼
- 😂 **搞笑反套路**：你以为你是勇者？不，你只是个修电脑的
- 🎭 **多结局**：3 种不同结局，取决于你的选择
- 🖱️ **简单操作**：方向键移动，回车/空格互动
